# Carbon leakage from the EU's Aluminium sector

This repository provides the code for the conference paper ["EU Emission Trading and Aluminium Imports: Evidence for Carbon Leakage"](https://www.researchgate.net/profile/Jacob-Thraen/publication/389549883_EU_Emission_Trading_and_Aluminium_Imports_Evidence_for_Carbon_Leakage/links/67c72843f5cb8f70d5c7a650/EU-Emission-Trading-and-Aluminium-Imports-Evidence-for-Carbon-Leakage.pdf), which was presented at the 18th IAEE European Conference in Milan (2023). It employs a linear regression to predict net aluminium imports into the EU with the average price of emission allowances in that month. A range of additional explanatory variables is also included, e.g. domestic industrial demand. Several serial correlation measures are trialled, including Hildreth-Lu, Prais-Winsten, and Cochrane-Orcutt.
